## 🖥️ Features
- **Interactive CLI Terminal**: Type real commands to navigate through my bio, education, skills, and projects.
- **Customizable Phosphor Themes**: Cycle through different retro themes like classic green, amber glow, cyber cyan, and matrix white.
- **Retro Sound Synthesis**: Enjoy authentic keyboard clicks, floppy drive seeks, and hum sound effects (can be toggled).
- **Matrix Code Rain**: Fully interactive matrix rain effect on command.
- **Teardown Outro**: Scroll down for an exploded, floating component outro of the terminal system.

## 🚀 Running Locally

### Prerequisites
- Node.js (version 18+)

### Steps
1. **Clone the repository**:
   ```bash
   git clone https://github.com/workwithme2003-cmyk/PORTFOLIO.git
   cd PORTFOLIO
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the local development server**:
   ```bash
   npm run dev
   ```
   Open `http://localhost:3000` in your browser.

4. **Build for production**:
   ```bash
   npm run build
   ```
